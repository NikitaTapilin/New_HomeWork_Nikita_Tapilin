## Списки

Чтобы добавить ненумерованные списки необходимы выделить пункты звёздочками (*). Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки необходимы просто пронумеровать. Например вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
